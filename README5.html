Jvav对象和类

JVAV对象和类

是的，类只有主类和jvav提供的沙雕类！
而对象也一样，你只能new自己或者jvav提供的沙雕类
jvav永远遵循操蛋写法

类

类是可以创建对象的模板

//FuckYouWorld就是一个类，主不过他是主类罢了
public class FuckYouWorld {
    public static void main() {
    }
}

对象

上文所述，只能new的也就本身和jvav提供的对象 对象是使用new创建的一个对象

/*由于jvav取消了形参和实参的机制
你不能传参
*/
String str = new String();
