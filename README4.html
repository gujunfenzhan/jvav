Jvav基础语法

JVAV基础语法

一个Jvav程序只能实例自己，不能多个对象组合

  • 对象：类的实例
  • 类：jvav程序只允许有一个类，也就是主类
  • 方法:tan90° jvav取消了复杂的方法，让新手远离形参实参，这能给新手学习jvav节省大部分的时间
  • 实例变量：tan90° jvav也取消了复杂的实例变量

    第一个FuckYouWorld程序

    学会jvav，从祖安程序开始
    public class FuckYouWorld {
  /* 第一个Jvav程序
   * 它将打印字符串 FuckWorldWorld
   */
  public static void main() {
      System.out.println("FuckWorldWorld"); //龙门粗口
  }
}

    运行jvav

    新建文本，输入上面代码，保存(ctrl+s),把文件名改成FuckYouWorld.jvav,打开cmd,CD到FuckYouWorld.jvav当前路径，输入jvavc FuckYouWorld.jvav即可完成对代码的编译,接着输入jvav FuckYouWorld即可运行程序
    输出内容:
    >FuckYouWorld
